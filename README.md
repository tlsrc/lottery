lottery
=======

Simple command line lottery as a training project.

How to build
------------
1. *clone* the repo
2. *mvn package*

How to run
----------
 *java -jar target/lottery-1.0.jar*

How to use
----------
- *achat X* to buy a ticket as X
- *tirage* to pick the winners
- *gagnants* to display the winners
